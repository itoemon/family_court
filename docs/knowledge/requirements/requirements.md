# 要件書: 共通ヘッダー・フッター

## 機能要件

### ヘッダー
- 全ページに表示される共通ヘッダーを実装する
- アプリ名（「家庭裁判所」）をロゴ代わりにテキストで表示する
- ログイン済みの場合、右側にプロフィールリンク（またはアイコン）とログアウトボタンを表示する
- 未ログインの場合、右側にログイン・サインアップへのリンクを表示する
- ヘッダーのロゴ部分をクリックするとトップページ（`/`）へ遷移する

### フッター
- 全ページに表示される共通フッターを実装する
- コピーライト表記を含む（例: © 2026 家庭裁判所）
- シンプルで目立ちすぎないデザインにする

## 非機能要件

### デザイン
- このアプリは恋人・夫婦・家族が使う「話し合いの場」であるため、温かみのある・柔らかい雰囲気を大切にする
- 対立や緊張感を煽るような配色・表現は避ける
- Tailwind CSS を使用する
- モバイルファースト・レスポンシブ対応

### 実装
- Next.js App Router の `app/layout.tsx` に組み込む
- ヘッダー・フッターはそれぞれ `app/components/Header.tsx`、`app/components/Footer.tsx` として分離する
- ログイン状態の取得は Supabase の `createSessionClient()` を使用する（サーバーコンポーネント）

## 対象外
- ナビゲーションメニューの実装（ヘッダーロゴ + 認証状態表示のみ）
- 多言語対応
