# archive/

エージェント (アーキ・ビルド・テスタ・オーディ) の参照対象外。

`scripts/rotate_logs.sh` が直近 N 件 (デフォルト 3) を超えた `audit-log/` と `test-log/` の古いログをここに退避する。リードがあとから振り返るための保管庫であり、新しいエージェント呼び出しのコンテキストには載せない。

## ディレクトリ構成

- `audit-log/` — 退避された監査ログ
- `test-log/` — 退避されたテストログ

## 復元

特定のログを再度アクティブにしたい場合は手動で `docs/knowledge/<name>/` に戻す。
