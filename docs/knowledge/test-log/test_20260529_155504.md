## 原因分析

1. **環境要件の不満足**
   - Next.js 16.2.6（`package.json` に記載）は Node.js 20.9.0 以上を必須とする
   - 現在のシステム Node.js は 18.20.2
   - volta に `package.json` での Node.js pin 設定がない

2. **dev サーバー起動不可**
   - `npm run dev` は即座に Node.js バージョンチェックで失敗
   - localhost:3000 に接続できず、E2E テスト（Playwright）実行不可

3. **テスト実行不可**
   - dev サーバーが起動していないため、Playwright テストが localhost:3000 を叩けない
   - **全テストシナリオ（CRITICAL + LOW バッチ検証）実施不可**
