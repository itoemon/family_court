## 失敗原因の分析

### 公開トグルボタン未検出の原因

**`/laws/[id]/page.tsx` の実装確認**:

```tsx
const isOwner = law.owner_id === user.id;

return (
  <main className="min-h-screen bg-stone-50">
    <div className="max-w-2xl mx-auto px-4 py-10 space-y-6">
      {/* ... */}
      {isOwner && (
        <VisibilityToggle lawId={lawId} isPublic={law.is_public} />
      )}
    </div>
  </main>
);
```

**想定される失敗パターン**:

1. **owner_id 判定の失敗**
   - `law.owner_id` が作成者の ID と一致していない
   - 実装側で INSERT 時に owner_id を正しく設定していない可能性

2. **DB クエリエラー（silent failure）**
   - `laws.is_public` 列が存在しない（migration 未適用） ← **これは修正した**
   - その他の列不足またはスキーマミスマッチ

3. **ページレンダリングエラー**
   - `/laws/[id]/page.tsx` のどこかで例外発生
   - Server Component がエラーでクラッシュ → 404 相当

4. **テストの前提ミス**
   - テストが作成した法律と、実装の動作に乖離
   - 例: 作成直後の owner_id が正しく設定されていない
