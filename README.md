# igiari

大切な人とおだやかに話し合うための場所。恋人・夫婦・家族が裁判形式で意見を交わし、AI 裁判官が判決を下す Web アプリです。

> 旧名: `family_court`。PR #17 (FEAT-001) でブランド名を `igiari` にリネームした。Vercel プロジェクト名と `package.json` の `name` フィールドは新名 `igiari` に揃えている。GitHub リポジトリ名（および本リポジトリの作業ディレクトリ名）は旧名 `family_court` のまま残っているため、リポジトリ URL（`github.com/itoemon/family_court`）と表示名（igiari）が一致しない点に注意。

## Getting Started

```bash
npm run dev
```

[http://localhost:3000](http://localhost:3000) をブラウザで開いてください。

## Tech Stack

- [Next.js](https://nextjs.org) (App Router)
- [Supabase](https://supabase.com) (Auth / DB)
- [Anthropic Claude API](https://www.anthropic.com)
- [Tailwind CSS v4](https://tailwindcss.com)

## Deploy

[Vercel](https://vercel.com) へのデプロイを推奨します。
