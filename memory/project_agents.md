## エージェント一覧

| ニックネーム | 役割 | 実体 |
|---|---|---|
| **リード** | 要件定義・方針決定・PM | Claude Code（対話チャット） |
| **アーキ** | 設計書作成 | Claude CLI エージェント |
| **ビルド** | コード実装 | Claude CLI エージェント |
| **オーディ** | 品質監査・コードレビュー | Claude CLI エージェント |
| **コパ** | PR最終レビュー | GitHub Copilot |

**Why:** 打ちやすさ重視でカタカナ短縮形に統一。役割が直感的に伝わる命名。

**How to apply:** 会話中でエージェントを指すときはこのニックネームを使う。ドキュメントやシェルスクリプトにも同じ名称を使用する。
